# MvpDemo
#介绍
参考Google官方MVP架构写的Demo，并加入了一些Android5.0以及动画的效果，作为一个Android菜鸟代码还有很多不足之处，如有问题欢迎指出。
应用中所用的API是通过参照其他项目源码得到，如有侵权请及时联系我删除。

因为参考了很多开源项目以及博客，在代码注释中会标有原文链接。

#注意
因为一些动画的需要，请保持RecyclerView 23.1.0或者更高
